# 러쉬(LUSH) – 웹사이트 리디자인 프로젝트

## 1. 프로젝트 소개

이 프로젝트는 **러쉬(LUSH) 코스메틱 브랜드를 콘셉트로 한 반응형 쇼핑몰 웹사이트**입니다.

- 러쉬 스타일에 맞게 상품 데이터와 페이지 콘텐츠를 구성하고,
- 메인, 상품 목록, 베스트, 장바구니, 마이페이지, 상품 상세 페이지까지
- HTML, SCSS, JavaScript로 화면 구성과 기본 동작을 구현한 프론트엔드 프로젝트입니다.

---

## 2. 사용 기술

- **HTML5** – 기본 레이아웃, 시맨틱 태그
- **CSS3 / SCSS** – 반응형 레이아웃, 공통 스타일, 그리드/플렉스
- **Vanilla JavaScript** – DOM 조작, 이벤트 처리, localStorage 사용
- **Bootstrap Icons** – 아이콘 표시

---

## 3. 페이지 구성

- `index.html` – 메인 페이지
  - 메인 배너 슬라이드
  - 베스트 / 신상품 / 콜라보 상품 섹션
  - 상단 검색, 장바구니 아이콘

- `sub1.html` – 상품 목록 페이지
  - 전체 상품 리스트
  - (UI만 있는) 카테고리 / 가격 / 정렬 필터
  - 검색 결과 표시

- `sub2.html` – 베스트 상품 페이지
  - 베스트 상품만 따로 모아 출력
  - TOP3 강조 카드 + 나머지 리스트

- `sub3.html` – 장바구니 페이지
  - 담은 상품 목록, 수량 변경, 삭제
  - 총 금액 계산

- `sub4.html` – 마이페이지
  - 탭 UI (주문내역, 찜한 상품 등)
  - 찜한 상품 리스트 (localStorage 기반)

- `product-detail.html` – 상품 상세 페이지
  - 상품 이미지, 설명, 스펙, 리뷰 표시
  - 장바구니 담기 버튼

---

## 4. 주요 기능 정리

### 4-1. 상품 데이터 & 카드 렌더링

- `js/products-data.js`
  - `productsData.best`, `productsData.new`, `productsData.Collab` 형태로  
    **러쉬 실제 상품 느낌으로 더미 데이터 직접 작성**
  - 이미지 경로, 가격, 세일 여부, 설명, 스펙, 리뷰 등 포함

- `js/products.js`
  - `createProductCard(product)`  
    → 상품 객체를 받아서 카드 HTML 문자열로 변환
  - `loadProducts(containerId, products)`  
    → 특정 컨테이너에 상품 카드들 렌더링
  - `attachProductEvents()`  
    → 장바구니 버튼, 찜 버튼, 상세보기 버튼에 이벤트 연결

---

### 4-2. 상품 목록 / 검색

- **검색 기능**
  - 상단 검색창에서 입력 후 이동 시 `?search=검색어` 쿼리 파라미터 사용
  - `searchProducts(query)` 함수로
    - `productsData.best + new + Collab` 을 한 배열로 합치고
    - 제목에 검색어가 포함된 상품만 필터링

- **검색 결과 없음 처리**
  - 일치하는 상품이 없으면
  - “검색한 상품이 존재하지 않습니다” 안내 UI 표시

- **일반 상품 목록**
  - 검색어가 없을 경우,
  - 전체 상품을 모아서 일정 개수만 목록에 표시

---

### 4-3. 상품 상세 페이지

- URL의 `?id=상품ID`를 읽어서
- `findProductById(productId)`로 해당 상품 찾기
- 이미지, 제목, 가격, 설명, 스펙, 리뷰를 화면에 출력
- “장바구니 담기” 버튼 클릭 시 선택된 상품이 localStorage 장바구니에 저장

---

### 4-4. 장바구니 기능

- **저장 방식**
  - `localStorage`에 `cartItems` 배열로 저장
  - 예시 구조:
    ```json
    [
      {
        "id": 1,
        "title": "슬리피 샤워 젤",
        "price": 18000,
        "quantity": 2,
        "image": "images/sleepy-shower-gel-01.png"
      }
    ]
    ```

- **기능**
  - 장바구니 추가 / 삭제
  - 수량 증가 / 감소
  - 총 금액 자동 계산
  - 헤더 상단 장바구니 아이콘 옆 숫자 표시 (`updateCartCount()`)

## 6. 프로젝트를 하면서 배운 점

- **HTML/CSS**
  - 시맨틱 태그(`<header>`, `<main>`, `<footer>` 등)를 왜 쓰는지 이해하게 됨
  - Flex, Grid를 써서 반응형 레이아웃을 잡는 연습
  - SCSS로 색상/폰트/간격 등을 변수로 관리하면 유지보수가 쉬워진다는 점

- **JavaScript**
  - 상품 데이터를 **객체/배열**로 관리하고,  
    그걸 `map()`으로 돌려서 HTML을 만들어 넣는 패턴에 익숙해짐
  - DOM에서 요소를 찾고(`querySelector`), 이벤트를 걸고,  
    필요한 데이터를 꺼내서 사용하는 흐름을 연습
  - `localStorage`에 JSON 형태로 데이터를 저장/로드하는 방법 익힘

- **기타**
  - 코드를 하나씩 뜯어보면서  
    “이 함수가 어떤 역할을 하는지” 이해하려고 노력하는 경험
  - 버그가 났을 때 `console.log()`로 하나씩 찍어보면서  
    원인을 찾는 습관을 들이기 시작함
